My Background

The engineer behind the code.

I started with a deep curiosity for how things work — not just on the surface, but at the system level. That curiosity led me to Java and PHP, where I learned to think in structures: data flows, request cycles, and the invisible architecture that keeps applications alive.

Over time, I expanded into Python for AI/ML workflows, React and JavaScript for modern front-end experiences, and a sharp focus on data tooling with Excel dashboards and structured documentation. Each stack added a new lens for solving problems.

Today, I work across the full spectrum — from building an AI-powered SaaS Reply Studio (PersonaAI) to engineering a live university timetable system (LASU Diploma Timetable) to shipping a full e-commerce platform for premium wigs (Glam Wig Studio). Every project is a chance to go deeper, build better, and deliver something that genuinely works.

Based in Lagos, Nigeria, I'm building for a global audience — one resilient system at a time.
